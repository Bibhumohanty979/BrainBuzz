# BrainBuzz : Interactive Quiz Application

Intern ID: CITS6090

Full Name: Bibhu Kalyan Mohanty

No. of Weeks: 4 Weeks

Project Name: BrainBuzz Interactive Quiz Application

BrainBuzz – A modern and interactive quiz application with multiple quiz categories, difficulty levels, multiple-choice questions, timer, instant answer feedback, score tracking, progress tracking, high scores, and answer review features. Built using HTML, CSS, and JavaScript. Users can select a quiz category and difficulty level, answer timed questions, receive instant feedback, view their final score, and review their answers after completing the quiz.

# Features

📚 Multiple Quiz Categories

🎯 Difficulty Level Selection

❓ Multiple-Choice Questions

⏱️ Quiz Timer

📊 Question Progress Tracking

✅ Instant Answer Feedback

🏆 Score & Points Tracking

🥇 High Score Tracking

📈 Final Quiz Results

📋 Answer Review

🔄 Play Again Functionality

🌙 Light & Dark Mode

📱 Responsive Design

💾 Local Storage Support

# 🛠️ Technologies Used

HTML5

CSS3

JavaScript

# 📌 Project Purpose

This project demonstrates the development of an interactive and responsive quiz application using frontend web technologies. Users can test their knowledge across different categories and difficulty levels while receiving instant feedback, tracking their scores, and reviewing their answers.
